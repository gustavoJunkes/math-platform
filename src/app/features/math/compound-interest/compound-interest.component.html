Como funciona o cálculo de juros compostos

A calculadora acima é capaz de calcular juros compostos. A seguir veremos como funciona o cálculo de juros compostos.

O cálculo de juros compostos é aquele em que a taxa de juros, predefinida ou não, atua sobre o montante a cada período. Isso significa que os juros do segundo período serão calculados sobre o valor total, incluindo o valor dos juros acumulado dos períodos anteriores. Isso representa um crescimento exponencial do montante, o que é favorável para investimentos, mas indesejável para dívidas.

A fórmula usada para calcular juros compostos é a seguinte:

M = C x (1 + i)^t

Onde:

  • M = montante
  • C = capital inicial
  • i = taxa de juros
  • t = tempo/duração da aplicação

Como calcular juros Simples

A calculadora acima é capaz de calcular juros simples. A seguir veremos como o cálculo funciona.

O cálculo de juros simples é aquele em que a taxa de juros, um valor predefinido, só atua sobre o valor inicial. A fórmula que usamos para calcular juros simples é:

valor final = valor inicial + juros sobre valor inicial + valor por período * número de períodos
